# Development


## Reflectie Development

Ik heb dit semester aan het eind wat development werk moeten doen, omdat ik graag mijn skills wilde laten zien. In het begin heb ik weinig aandacht geschonken aan dit leerdoel, omdat ik dacht dat we dit zouden ontwikkelen, doormiddel van onze proftaak. Achteraf heb ik er spijt van gehad dat ik er zo laat mee ben gaan werken, omdat ik mezelf niet helemaal heb kunnen laten zien. Toch heb ik wel veel kennis opgedaan door mijn passion project die ik heb afgerond. Ook ben ik alle workshops doorgelopen, maar zijn ze niet allemaal gelukt. Later wil ik niks met development doen, dus kan het ook zo zijn geweest dat ik daarom minder aandacht eraan gaf.
